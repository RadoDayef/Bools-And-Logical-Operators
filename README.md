## What's Inside?

2. **Logical Operators:**
    * `!`: Logical NOT -> !true = false, !false = true.
    * `||`: Logical OR -> It returns true if at least one condition is true, It only returns false when both conditions are false.
    * `&&`: Logical AND -> It returns true only if both conditions are true, If even one condition is false, the result becomes false.

### OR Truth Table

| A     | B     | A OR B |
| ----- | ----- |--------|
| true  | true  | true   |
| true  | false | true   |
| false | true  | true   |
| false | false | false  |

### AND Truth Table
| A     | B     | A AND B |
| ----- | ----- |---------|
| true  | true  | true    |
| true  | false | false   |
| false | true  | false   |
| false | false | false   |

### NOT Truth Table
| A      | NOT A |
|--------|-------|
| !true  | false |
| !false | true  |
